BoubsDaara is a front-end educational web platform designed for students and teachers.  
It allows users to authenticate, select their class and course, and then access or upload academic resources such as quizzes, midterms,Lectures and final exams.

This project was built as part of a **Web Development class project**.

---

##  Features

-  User authentication (Firebase Auth)
-  Dashboard with class and semester selection
-  Course pages with categorized content:
  - Quiz
  - Midterm
  - Lecture
  - Final Exam
- File upload system (front-end based)
- View uploaded files per category
- Clean and responsive UI (HTML & CSS)

- **GitHub Pages** (Deployment)
